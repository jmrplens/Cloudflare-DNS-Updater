## Características

<CardGrid stagger>
	<Card title="Actualizaciones por lotes" icon="rocket">
		Todos los cambios de registros se envían en una sola llamada a la API de
		Cloudflare, con paginación automática para zonas grandes.
	</Card>
	<Card title="Detección de IP inteligente" icon="magnifier">
		Lectura de direcciones IPv6 globales estables desde la interfaz local, con
		varios servicios externos de respaldo para IPv4 e IPv6.
	</Card>
	<Card title="Multiplataforma" icon="laptop">
		Funciona donde funcione Bash — Linux, macOS y Windows — y publica binarios
		autónomos que incluyen bash, curl y jq.
	</Card>
	<Card title="Notificaciones" icon="email">
		Avisos opcionales por Telegram y Discord cuando cambian los registros.
	</Card>
